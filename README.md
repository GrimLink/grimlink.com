# Grimlink.com

The personal website and digital garden of Sean van Zuidam.

Built with a focus on simplicity, performance, and modularity, this site serves as a central hub for my professional work, technical deep dives, and personal interests.

## Tech Stack

- **Framework:** [Astro](https://astro.build/) (Static Site Generation)
- **Styling:** [Fylgja CSS](https://fylgja.dev/) (Modular CSS framework)

## Features

- **Digital Garden:** A place for evolving thoughts, from Magento 2 and Hyvä Themes to music and gardening.
- **Social Archive:** A unified home for technical tips and posts originally shared across various social platforms.
- **Performance First:** Zero-JS by default, leveraging Astro's "HTML-first" architecture.
- **Modular Design:** Styled using a surgical selection of Fylgja CSS modules.

## Development

### Prerequisites

- Node.js `^22.12.0`
- npm

### Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Commands

| Command | Action |
| :--- | :--- |
| `npm run dev` | Starts local dev server at `localhost:4321` |
| `npm run build` | Build your static site to `dist/` |
| `npm run preview` | Preview your build locally |
